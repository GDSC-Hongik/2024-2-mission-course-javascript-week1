# HTML5
- html은 웹페이지를 기술하기 위한 마크업 언어이다.
- 웹페이지의 내용과 구조를 담당하는 언어

```html
<!DOCTYPE html>
<html>
    <head></head>
    <body></body>
</html>
```
- 이런식으로 구성됨
<p></p> 이게 요소임

## stype
```html
<!DOCTYPE html>
<html>
    <head>
        <style>
            body{
                background-color: yellow;
            }
        </style>
    </head>
    <body>
        <p>a</p>
    </body>
</html>
```
- 이런 식으로 스타일 넣을 수 있음 

## 형태
bold <b></b>
